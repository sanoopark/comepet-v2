## ⭐️ 소개

- ComePet은 실종된 반려동물을 찾고 유기동물을 입양하도록 돕는 서비스입니다.
- 기존 유사 서비스의 복잡한 UI를 단순화해 사용성을 높였습니다.
- [프로그래머스 데브코스에서 진행한 팀 프로젝트](https://github.com/prgrms-web-devcourse/Team_i6_comepet_FE)를 기술적으로 개선하는 개인 프로젝트입니다.
